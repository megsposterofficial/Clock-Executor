# Clock Executor



##### Clock Executor, is a Roblox Exploit Executor that allows for players to execute scripts an use them in game.



To use it, go to the Releases tab, ```https://github.com/megsposterofficial/Clock-Executor/releases/tag/v1.0```.



Download the any of the versions that are listed, **`Recommended to get the latest version`.**



This Executor is powered by Xeno API.



# Build Options

##### To build this project you must have Visual Studio 2022 or later to build.

##### Download the Project via github, then open up the ClockEXEC.sln file.

##### Then find the Build button and you will get the clean polished Clock Executor.
