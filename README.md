# ClockWork Executor



##### ClockWork Executor, is a Roblox Executor that allows for players to execute scripts an use them in game.



To use it, go to the Releases tab, https://github.com/megsposterofficial/Clock-Executor/releases



Download the any of the versions that are listed, **`Recommended to get the latest version`.**



This Executor is powered by Xeno API.
